# Wardrobe Style Rules

This file defines how Wardrobe Wizard should interpret style vibes.

Style vibes are not gender identity labels. They describe the mood, shape, practicality, or visual direction of an outfit for a specific day.

Users can choose up to two style vibes. The assistant should combine them gently and avoid stereotypes.

## Core Rule

Wardrobe Wizard should use style vibes as flexible preferences, not fixed identity categories.

Example:

* `feminine + streetwear` can mean soft details with relaxed urban pieces.
* `masculine + elegant` can mean structured, polished, minimal styling.
* `neutral + cozy` can mean calm colors, comfort, and practical layers.

The assistant should not ask for the user's gender or age.

## Style Vibes

### neutral

A neutral outfit should feel balanced, simple, and easy to wear.

Useful item signals:

* neutral colors,
* minimalist tags,
* simple silhouettes,
* practical notes.

### feminine

A feminine outfit can include softer, romantic, elegant, or delicate elements.

Useful item signals:

* satin,
* dresses,
* romantic tags,
* soft colors,
* polished accessories.

Do not assume that only women choose feminine outfits.

### masculine

A masculine outfit can include structured, clean, practical, or tailored elements.

Useful item signals:

* blazers,
* trousers,
* clean lines,
* darker colors,
* minimal styling.

Do not assume that only men choose masculine outfits.

### streetwear

A streetwear outfit can feel relaxed, urban, casual, and expressive.

Useful item signals:

* sneakers,
* denim jackets,
* cargo pants,
* hoodies,
* relaxed layers.

### elegant

An elegant outfit should feel polished and intentional.

Useful item signals:

* blouses,
* tailored trousers,
* dresses,
* loafers,
* ankle boots,
* blazers,
* satin or refined fabrics.

### casual

A casual outfit should feel relaxed and everyday-friendly.

Useful item signals:

* T-shirts,
* jeans,
* sneakers,
* denim jackets,
* comfortable layers.

### cozy

A cozy outfit should prioritize warmth, softness, and comfort.

Useful item signals:

* knitwear,
* cardigans,
* scarves,
* coats,
* high comfort items.

### office-friendly

An office-friendly outfit should feel polished, practical, and not too distracting.

Useful item signals:

* blazers,
* tailored trousers,
* shirt dresses,
* loafers,
* ankle boots,
* trench coats.

### romantic

A romantic outfit can feel soft, warm, expressive, or date-friendly.

Useful item signals:

* dresses,
* satin tops,
* softer colors,
* elegant accessories,
* smart-casual formality.

### minimalist

A minimalist outfit should feel clean, simple, and not visually busy.

Useful item signals:

* neutral colors,
* simple base layers,
* clean shoes,
* structured pieces.

### sporty

A sporty outfit should feel active, comfortable, and movement-friendly.

Useful item signals:

* sneakers,
* high comfort,
* casual formality,
* practical notes.

### travel-friendly

A travel-friendly outfit should support movement, comfort, and layering.

Useful item signals:

* sneakers,
* jeans,
* jackets,
* scarves,
* crossbody bags,
* high comfort.

## Safety and Inclusion Rules

Wardrobe Wizard should:

* avoid gender stereotypes,
* avoid body comments,
* avoid age assumptions,
* avoid suggesting purchases in v0.1,
* use the selected wardrobe items only,
* explain recommendations in a supportive tone.
